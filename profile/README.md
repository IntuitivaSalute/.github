# 🏥 Intuitiva Salute

Benvenuti nell'organizzazione GitHub ufficiale di **Intuitiva Salute**. La nostra missione è semplificare l'accesso alle cure e la gestione della salute attraverso soluzioni digitali intuitive, sicure e all'avanguardia.

## 🚀 I Nostri Progetti

Gestiamo due ecosistemi principali che lavorano in sinergia per offrire la migliore esperienza possibile a pazienti e professionisti:

### 1. [Sito Istituzionale](https://intuitivasalute.it)
*Repository: `intuitiva-site`*
La nostra vetrina pubblica. Si concentra su:
- Presentazione dei servizi e della filosofia aziendale.
- Blog informativo e risorse per la salute.
- Lead generation e contatti.

### 2. [Web Application](https://app.intuitivasalute.it)
*Repository: `intuitiva-app`*
Il cuore pulsante del nostro servizio. Una piattaforma SaaS dedicata a:
- Prenotazione e gestione appuntamenti.
- Telemedicina e consulti online.
- Dashboard per il monitoraggio dei parametri vitali e cartella clinica digitale.

---

## 🛠 Stack Tecnologico

Utilizziamo tecnologie moderne per garantire scalabilità, performance e sicurezza:

- **Frontend:** React.js / Next.js, Tailwind CSS, TypeScript.
- **Backend:** Node.js / Python, Architettura a Microservizi.
- **Database:** PostgreSQL, Redis.
- **Infrastruttura:** Docker, Kubernetes, CI/CD tramite GitHub Actions.
- **Sicurezza:** Conformità GDPR, Crittografia AES-256, Standard HL7/FHIR.

## 🤝 Collaborazione

Se sei un membro del team o un collaboratore esterno:

1. **Branching Strategy:** Utilizziamo il modello GitFlow (`main`, `develop`, `feature/*`).
2. **Pull Requests:** Tutte le PR richiedono almeno una review tecnica prima del merge.
3. **Coding Standards:** Seguiamo le linee guida ESLint/Prettier definite nei singoli repo.
